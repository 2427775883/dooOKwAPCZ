# Django航空售票系统源码分享

## 核心功能
- 登陆页面：账户登录、身份证登录（仅限用户）、用户注册（仅限用户）、记住密码功能
- 航班管理页面：查看航班列表、多功能筛选（航班号、出发城市、到达城市、日期）、添加航班功能（数据入数据库）、发送延误通知功能
- 订单管理页面：查看订单列表、根据用户名称模糊查询
- 机票管理页面：查看所有机票、展示航班通知（目前只有延误通知）
- 用户管理页面：查询、添加、编辑用户信息（仅限手机号）
- 购票大厅：查看航班列表、多功能筛选、购票
- 购票记录页面：查看订单记录、查看机票、查看航班通知、退票功能
- 个人中心页面：查看、修改个人信息

## 适用场景
- 航空公司内部售票系统
- 网上购票平台
- 旅行社预订系统

## 技术特点
- 基于Django框架，使用MySQL数据库
- 使用HTML、CSS和JavaScript进行前端开发
- 支持用户和管理员权限管理
- 提供航班查询、订单管理、机票购买等功能
